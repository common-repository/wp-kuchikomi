This plugin add a 5-point rating system for your wordpress, and can also accept ratings from users.

== Description ==

It is a plugin which can set and display a 5-point rating for posting. There are few setting items, it can be used simply.

Features include:

- A 5-point rating can be set for each item for posting
- Display rating by star in post
- Ranking display widget by rating points
- User ratings can be accepted from comments

And more

- Display rating by user average 
- Support other posttypes
- Items can be set variably
- Shortcode that can display ratings in any place
- Shortcode that can display ranking in any place


== Installation ==

1. `wp-kuchikomi` フォルダを `/wp-content/plugins/` ディレクトリの配下にアップロードします
1. 'プラグイン' メニューからプラグインを有効化します
